# 業務術語詞彙表 · Glossary

> **用途：** 統一本系統所有 UI 文字、文件、日誌訊息的中文用詞，避免同一概念出現多種譯法。
>
> **原則：**
> - UI 文字：使用「繁中欄」的用字
> - 代碼識別字 / DB 欄位：使用「English」欄
> - 簡體 → 繁體：嚴禁出現簡體字。如發現請立即修正。

---

## 詞彙更新流程

當 HR 或法律顧問提出更準確的用詞時：

1. 在此表新增 / 修改條目
2. 用 `git grep` 找出所有使用舊詞的地方
3. 一次性替換，commit message：`chore(glossary): unify "XXX" → "YYY"`
4. PR review 時再次確認無殘留
